# 明代會試《春秋》義搭股統計分析

## 概述

本倉庫為博士論文《明代科舉春秋義研究》第四章之統計分析代碼與數據，包含以下分析：

1. **獨立樣本t檢定**：以成化十四年（1478）為分界，驗證前後期搭股數量的差異顯著性。
2. **效果量計算**：Cohen's d、Hedges' g、η²（Eta-squared）。
3. **變點檢測**：以五種演算法（PELT、二元分割、由下而上、動態規劃、CUSUM）定位時間序列的結構斷裂點。
4. **BIC模型選擇**：確定最佳變點數目。

## 資料說明

`data/huishi_data.csv` 欄位：

| 欄位 | 說明 |
|------|------|
| `era_name` | 科年（年號紀年） |
| `western_year` | 西元年 |
| `q1_stocks` ~ `q4_stocks` | 四題各自的搭股數（空值表示該題佚失） |
| `n_questions` | 該科有效題數 |
| `avg_stocks` | 四題平均搭股數 |
| `dati_rate` | 搭股率（搭股數>1的題目佔比） |

資料來源：明代會試錄、《皇明貢舉考》、《梅林臆見》等。詳見論文第四章第二節。

## 主要分析結果

### t檢定與效果量

以成化十四年（1478）為分界：
- 前期平均搭股：約3.4，後期約1.8
- Cohen's d ≈ 1.41（極大效果，超過大效果門檻1.8倍）
- η² ≈ 0.42（解釋41.6%的變異）

### 變點檢測

五種方法一致指向成化年間（1466-1481）為結構斷裂區間。BIC最佳模型為k=2，兩個變點分別位於：
- 成化十七年（1481）
- 嘉靖五年（1526）

此一結果與t檢定劃定的成化十四年（1478）僅差一科，構成獨立的交叉驗證。
